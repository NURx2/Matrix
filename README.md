# Matrix

C++17 header which simplifies the use of matrices.

## Usage examples

#### Including the header file

```cpp
#include "matrix.h"
```

### Basic

+ creating a matrix

```cpp
nur::Matrix<int> m{
    {
        {
            4, 4,
        },
        {
            6, 5,
        },
    }
};
```

+ element access using `operator []`
